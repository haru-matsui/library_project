# Foliant — электронная библиотека

Foliant — веб-приложение на Flask для размещения, поиска и чтения книг в формате FB2. Администраторы управляют каталогом, пользователи читают и скачивают книги, а также сохраняют избранное. Доступен REST API с базовой авторизацией.

## Возможности
- Администратор: загрузка FB2, автоматическое извлечение метаданных и обложек, удаление книг.
- Пользователь: регистрация/вход, поиск по названию и автору, чтение в браузере, скачивание FB2, избранное.
- API: список книг и скачивание по Basic Auth.

## Стек
- Python 3.9+
- Flask, Flask-Login, Flask-RESTful, SQLAlchemy
- SQLite
- Bootstrap 5, HTMX

## Быстрый старт
1. Перейдите в каталог приложения: `cd readb`
2. Создайте виртуальное окружение и установите зависимости:
   - `python -m venv .venv`
   - `source .venv/bin/activate` (Windows: `.venv\Scripts\activate`)
   - `pip install -r requirements.txt`
3. Запустите приложение: `python main.py`
4. Откройте в браузере: http://127.0.0.1:5000

## Администрирование
- Права администратора назначаются пользователям с логинами `admin1` и `admin2`.
- Зарегистрируйте этих пользователей и перезапустите приложение, чтобы роль администратора активировалась.
- Альтернатива — установить `is_admin = True` вручную в базе данных.

## API
> Все API-эндпоинты используют HTTP Basic Auth (логин/пароль пользователя).

- `GET /api/books` — список книг в JSON.
- `GET /api/books/<id>` — скачать FB2 по id.
- `GET /api/users` — список пользователей (доступен при активной веб-сессии).

Пример запроса:
```bash
curl -u username:password http://127.0.0.1:5000/api/books
```

## Структура проекта
- `readb/main.py` — точка входа и маршруты.
- `readb/api/` — REST-ресурсы.
- `readb/data/` — модели и работа с БД.
- `readb/forms/` — формы.
- `readb/templates/` — HTML-шаблоны.
- `readb/db/users.db` — база данных SQLite.
- `readb/booky/` — каталог для загруженных книг и обложек (создаётся при запуске).

## Примечания
- Поддерживается формат FB2; обложка извлекается из файла книги автоматически.
