Техническое задание (ТЗ) для создания сайта библиотеки

1. Назначение проекта
Разработка веб-приложения для электронной библиотеки, позволяющего:
	Администраторам управлять каталогом книг.
	Пользователям просматривать и скачивать книги после регистрации.

2. Функциональные требования
2.1. Для администратора:
	Вход в систему через логин и пароль.
	Добавление новых книг (название, автор, описание, обложка, файл в формате PDF/EPUB).
	Редактирование и удаление существующих книг.

2.2. Для пользователей:
	Регистрация и авторизация.
	Просмотр списка доступных книг с возможностью сортировки и поиска.
	Скачивание книг (доступно только авторизованным пользователям).

2.3. API:
	Получение списка книг в формате JSON (GET /api/books).
	Скачивание книги по ID (GET /api/books/{id}/download).

3. Техническая реализация
	Frontend: HTML, CSS, JavaScript
	Backend: Flask (Python) с REST-маршрутами.
	База данных: SQLite (для простоты разработки).

4. Интерфейс
	Минималистичный дизайн с удобной навигацией.
	Адаптивный макет (корректное отображение на ПК и мобильных устройствах).
	Формы для входа, регистрации и загрузки книг.

5. Этапы разработки
Проектирование:
	Создание макетов интерфейса.
	Определение структуры базы данных.

Реализация:
	Верстка сайта
	Разработка backend (авторизация, загрузка книг, API).

Тестирование:
	Проверка функционала на разных устройствах.
	Исправление ошибок.

7. Возможности расширения
В будущем можно добавить:
	Рейтинги и отзывы к книгам.
	Категории и теги для удобного поиска.
